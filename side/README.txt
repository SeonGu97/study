- 오늘은 디자인을 새로 만들었다. 웹 서칭중.. codepen에서 괜찮은 디자인을 보아서 카피를 해보았다.
디자인적인 부분만 따오고 만드는건 생각해 놓았던 레이아웃으로 만든다. 일단 먼저 상단에 데스크탑, 테블릿, 모바일을 나타내는
3개의 아이콘을 추가하고, 추후 아래에 엘리먼트를 만들어서 그 안에 콘텐츠를 담을건데 상단에 디바이스 버튼을 누르면
해당 디바이스의 크기로 메인 콘텐츠의 크기가 변경되도록 만들 예정이다. 이렇게 하면 물론 반응형으로 때워도 되지만, 귀찮게 화면을 줄였다 늘였다 하는
동작없이 버튼만 눌러도 반응형을 확인할 수 있도록 된다. 일단 오늘은 버튼이랑 애니메이션을 구현하고 내일은 사이즈가 변경되도록 기능을 추가해야겠다.

- 느낀점은 오늘 만들면서 항상 디자인적인 부분이 많이 부족했는데, 다른 사람들이 만들어 놓은 오픈소스를 경험하다 보면 정말 신기할 정도로
대단한 사람들이 많다고 느낀다. 나도 그렇게 되기 위해서 많은 사람들이 만들어 놓은 아이템들을 많이 경험해서 경험치를 늘이는 수 밖에 없을 것같다.
