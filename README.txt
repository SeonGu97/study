(21.10.22)
- 오늘은 버튼을 누르면 왼쪽, 오른쪽으로 넘어가는 슬라이더를 만들었다.
역시 기존에 쓰던 방식과는 다르게 애니메이션으로 만들었다. 넘어가는 부분이 부드럽고
깔끔하다. appendChild, removeChild를 이용해서 firstChild와를 지우고 다시 appendChild를
이용해서 지운 첫번째 자식을 마지막으로 붙여줬다. 반대로는 flex-direction을 이용해서
row와 row-reverse를 이용해서 위치를 바꾸어주었다. 결론적으로 만족하는 슬라이더가 완성됬다.

(21.10.26)
- 오늘은 2번째로 만든 애니메이션의 수정할 부분이 있어서 해결방법을 모색해 보았다.
문제는 슬라이더가 작동은 정상적으로 되지만 슬라이더를 넘기는 버튼을 연타하면 넘어가는 모션을

넘어가길 바랬는데 아직 애니메이션API는 익숙하지가 않아서 수정을 하지 못했다.

(21.10.27)

습득이 됬는데 애니메이션은 조금 깊게 공부를 해야 사용이 가능할 것 같아서
애니메이션을 공부하기로 했다.
