- 오늘은 네번째 슬라이더를 만들기 시작했다. 원래 네번째는 3개씩 보여지는 슬라이더를 만들려고 했는데 오랜만에 페이드인 페이드 아웃을 만들어 볼려고한다.
기존에 만드는 슬라이더는 5개의 슬라이더를 보여주려면 2개더 추가해서 7개를 생성 했어야 됬는데 이번 페이드인 페이드아웃 슬라이더는 5개만 만들면 될 것 같다.

(21.11.26)
- 오늘은 네번째 슬라이더 페이드인 페이드 아웃 슬라이더를 만들었다. 생각보다 너무 쉽게 만들어졌다. 그냥 이렇게 하면 작동이 되겠지 하는 생각에 만들었는데
막힘 없이 잘 만들어 졌다. 그전 슬라이더보다 난이도가 훨씬 낮았다. 그래서 다음은 무슨 슬라이더를 만들어야 할지 고민이다. 만들고 싶은 슬라이더는 엄청 많은데
무엇을 먼저 만들어야 할지 고민인 것이다. 이번에는 저번에 만들다 못한 세개씩 보여지는 슬라이더를 만들어 볼까한다. 세개의 슬라이더 아이템당 라디오 버튼도 하나씩
아니면 setinterval을 사용해서 특정 시간이 지나면 자동으로 넘어가는 슬라이더를 만들지 생각을 좀 해보고 만들어야겠다.

(21.11.29)
- 오늘은 다섯번째 슬라이더를 만들었다. 일단 네번째의 슬라이더와 가능은 똑같은데 거기에 타이머를 적용시켰다. 일정 시간이 지나면 슬라이더가 자동으로 넘어간다. 그리고
prev, next를 누르면 clear가 되고, 다시 실행되도록 만들었다. 이유는 시간이 멈추지 않고 계속 적용된다면, 슬라이더가 두번 넘어 갈 수도 있기 때문에 확실하게 멈추고 넘어가면
다시 타이머가 실행되도록 만들었다. 이번 슬라이더도 만들기 쉬웠다. 기존 네번째 슬라이더에 타이머만 추가하면 됬기 때문이다. 여섯번째는 조금더 어려운걸 만들어 봐야겠다.
